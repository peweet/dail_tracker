# Trademark Filing Checklist — "Dáil Tracker"

A turnkey runbook for registering the **"Dáil Tracker"** name as a trademark.
This backs the [`TRADEMARK_POLICY.md`](TRADEMARK_POLICY.md). *Not legal advice —
for the distinctiveness question (see Step 2) a trademark attorney is worth the
fee.*

> **You can use `™` right now**, today, with no registration — it signals you
> claim the name as an unregistered mark. Only `®` requires a granted
> registration. So start putting **Dáil Tracker™** on the site/README while the
> application is pending.

---

## Step 0 — Decide scope (5 min)

| Decision | Options | Suggested default |
|---|---|---|
| Territory | Ireland only (IPOI) · EU-wide (EUIPO) · both | **Start with Ireland (IPOI)** — cheap, covers your market; add EUIPO later if you expand |
| Mark type | Word mark · Figurative (logo) mark · both | **Word mark** first (protects the name in any font). Add a figurative mark if/when you have a logo — it's also a fallback if "Dáil Tracker" as words is judged too descriptive |
| Owner | Patrick Glynn (individual) · a company | Individual now; **reassign to a company if you incorporate** (a one-line assignment) |

## Step 1 — Pick the Nice classes (10 min)

Trademarks are registered against "Nice Classification" classes of goods/services.
For a civic-data software product, the relevant ones are:

- **Class 42** — *software as a service (SaaS); design/hosting of software;
  data analysis; providing online non-downloadable software.* **(core — file this)**
- **Class 9** — *downloadable/recorded software.* **(core — file this)**
- **Class 35** — *compilation/systemisation of information into databases;
  business data analysis.* (optional — relevant to the data-linkage angle)
- **Class 41** — *publishing; providing online publications.* (optional — only if
  you frame it as a publication/education service)

Each extra class adds a fee, so a lean filing is **Class 9 + 42**. Add 35 if
budget allows.

## Step 2 — Clearance search BEFORE filing (30 min)

Check nobody already holds a conflicting mark, and gauge whether "Dáil Tracker"
is registrable (government/descriptive terms can draw objections):

- [ ] **IPOI trademark search** — <https://www.ipoi.gov.ie/> → Trade Marks → search the register for "Dáil", "Tracker", "Dáil Tracker".
- [ ] **EUIPO / TMview** — <https://www.tmdn.org/tmview/> (searches IE, EU, and international marks at once).
- [ ] **Plain web + domain + CRO company-name** search for existing users of the name.
- [ ] **Distinctiveness gut-check:** "Tracker" is descriptive and "Dáil" is a
      government/official term — the combination is likely registrable but *could*
      face an objection that it's descriptive or that "Dáil" implies official
      status. This is the one point where a **trademark attorney's opinion**
      (often a free initial consult) is worth it. A **figurative/logo mark** is
      the standard mitigation if the plain word mark is challenged.

## Step 3 — File with IPOI (Ireland) (30–45 min)

- [ ] Create an account / log in at <https://www.ipoi.gov.ie/> (online filing).
- [ ] Start a **national trade mark application**.
- [ ] Enter the mark: **DÁIL TRACKER** (word mark). Mind the fada (á).
- [ ] Select the classes from Step 1 (9 + 42, optionally 35).
- [ ] Owner details: Patrick Glynn + address (or the company).
- [ ] Pay the fee. *(Fees change — check the current schedule on the IPOI site;
      online filing for a first class has historically been modest, with a
      smaller fee per additional class. Budget low hundreds of € for 2–3 classes.)*
- [ ] Save the application number and receipt.

## Step 4 — (Optional) File with EUIPO for EU-wide cover (later)

- [ ] <https://euipo.europa.eu/> → file an **EU trade mark (EUTM)**. One
      application covers all EU member states. Fee is higher (historically ~€850
      online for one class, with extra-class fees). Do this only if/when you
      expand beyond Ireland.

## Step 5 — After filing (the office handles the timeline)

Typical IPOI path, roughly **4–6 months** if unopposed:

1. **Examination** — IPOI checks for absolute grounds (e.g. descriptiveness — see
   Step 2). They may issue an examination report you must respond to.
2. **Publication** — the mark is published in the *Trade Marks Journal*.
3. **Opposition period** — **3 months** for third parties to object.
4. **Registration** — if unopposed (or you prevail), the mark registers. **Now
   you may use `®`.**

- [ ] Diarise any examination-report deadline (missing it abandons the application).
- [ ] Switch `™` → `®` once registered.
- [ ] Renew every **10 years** (set a calendar reminder).

## Step 6 — Wire the mark to the policy

- [ ] Add `™` (then `®`) to the name in the README, the app header, and the
      website.
- [ ] Confirm [`TRADEMARK_POLICY.md`](TRADEMARK_POLICY.md) is published/linked so
      the mark is backed by a stated usage policy.
- [ ] Record the registration number in this file once granted.
