# Dáil Tracker — Second-Pass Source Research Brief: Tangible Public-Record Data Sources

**Prepared:** 2026-06-05  
**Scope:** Second-pass source research for non-CBI public-record intelligence sources previously discussed.  
**Purpose:** Identify tangible, researchable public data sources: actual PDFs, CSV/XLSX datasets, HTML archives, decision pages, board minutes, meeting records, audit reports, performance reports, planning datasets, procurement datasets, and disclosure logs.

This is a source-discovery brief only. It does not prescribe architecture, phases, or implementation plans.

## 13. Suggested Claude research prompts

```text
For each source, determine:
- Is it HTML, PDF, CSV, XLSX, ArcGIS REST, GeoJSON, or another format?
- Is there a stable archive pattern?
- Is there pagination?
- Are document links discoverable from a landing page?
- Does the source expose dates, entities, monetary values, project names, or locations?
- Are records updated monthly, quarterly, annually, or irregularly?
- Are there privacy-sensitive fields?
- Which fields can be extracted reliably without OCR?
- Which pages require PDF table extraction?
- Which sources have clear source ownership and update cadence?
```
